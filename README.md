# Nutritional Awareness for All Drive

This project is created as part of a Community Services initiative to spread awareness about the importance of nutrition and healthy eating habits.

## 🌱 Project Overview
The aim of this project is to educate people about balanced diets, essential nutrients, and the impact of food choices on overall health. It also highlights problems like malnutrition and deficiency diseases.

## 💡 Features
- Information on balanced diet and nutrition
- Awareness about malnutrition and deficiency diseases
- Simple and interactive health calculators:
  - BMI Calculator
  - Daily Water Intake Calculator
  - Healthy Meal Guide
- Clean and user-friendly website design
